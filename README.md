# Samarth Lodging Website

A modern, responsive website for Samarth Lodging & Boarding located in Ketkawale, Taluka Purandar, District Pune.

## Features

- **Home Page**: Landing page with room previews, features, and nearby attractions
- **About Us**: Detailed information about the lodging services
- **Book Now**: Online booking system with payment options
- **Profile Page**: User profile management and booking history
- **Login System**: User authentication for booking management
- **Email Notifications**: Automatic booking confirmation emails (to be integrated)
- **WhatsApp Notifications**: Instant booking notifications via WhatsApp

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Netlify Functions
- **Database**: Firebase (to be integrated)
- **Hosting**: Netlify

## Getting Started

### Prerequisites

- Node.js installed
- GitHub account
- Netlify account
- Firebase account (for database)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Environment Variables

Create a `.env` file with the following variables:

```
FIREBASE_API_KEY=your-firebase-api-key
FIREBASE_AUTH_DOMAIN=your-firebase-auth-domain
FIREBASE_PROJECT_ID=your-firebase-project-id
FIREBASE_STORAGE_BUCKET=your-firebase-storage-bucket
FIREBASE_MESSAGING_SENDER_ID=your-firebase-messaging-sender-id
FIREBASE_APP_ID=your-firebase-app-id
WHATSAPP_API_KEY=your-whatsapp-api-key
```

### Deployment

1. Push code to GitHub
2. Go to Netlify dashboard
3. Import from Git
4. Add environment variables in Netlify (Firebase and WhatsApp credentials)
5. Deploy
